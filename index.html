CSS Notes

CSS is all about designing the Web and making your webpages look beautiful! Think of it this way:

  • HTML = content
  • CSS = presentation

EXTERNAL CSS:

Extenal CSS uses a separate style sheet that you link to in the head of the HTML document. External styles are the most commonly used and you can change the look of an entire web site by applying just one CSS file (.css file extension)!

EMBEDDED CSS:

Also called "internal" CSS. This type of styling will apply to elements on one HTML document. Apply it by using the <style> tags in the head of the html document. Embedded CSS takes precedence over external CSS.

INLINE CSS:

Lastly, lets make this line of text red using inline CSS. Inline CSS allows us to edit the styling of a single tag in our HTML document by using the style property. Inline CSS take precedence over the previous two types, so our text appears red!
